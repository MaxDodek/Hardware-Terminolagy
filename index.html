Input Device:

An electronic device that converts physical events - 0s and 1s. It allows users to interact with and control the computer. Ex. Mouse, microphone, etc. Go back to table of contents

Output Device:

Turns 0s and 1s to something physical (light, sound etc.) Ex. Flashlight, Printer, screen, etc.

Central Processing Unit (CPU):

The CPU is the brain of the computer. The CPU follows instructions from computer programs which tell it the data to process and how to process it - we wouldnt have any computer programs on computers without CPU. It is divided into two parts - the ALU and the Control Unit. With the ALU and Control unit, the computer can calculate extremely complicated things.

Logic Gates:

All processors are made of billions of Logic Gates that work together. Logic Gates are made up of transistors, and other electronic components such as capacitors and resistors. They simulate boolean logic (or, and, not) with electricity.

Random Access Memory (RAM):

RAM contains running programs and data that you are currently working on and is short term memory. It is extremely fast to access things,however it is temporary and needs power, or the memory can be erased if the computer turns off.
